Savings Calculator Widget - Development Preview

This is how your widget will appear when embedded in Webflow.

Usage in Webflow:

Use a Code Embed element with the following HTML:

<div id="data-savings-calculator"></div>
<script src="https://cdn.jsdelivr.net/gh/BEPP-AB/bepp-savings-calculator-widget@main/dist/savings-calculator-widget.iife.js"></script>

That's it! The widget will automatically load and render in the div. The Code Embed block contains both the container div and the script, making it a single-step integration.
